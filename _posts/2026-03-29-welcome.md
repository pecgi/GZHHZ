欢迎来到你的个人阅读汇总站！🎉

## 这是什么？

这是一个帮你汇总、整理微信公众号文章的阅读站。所有文章按日期自动归类，支持标签筛选和全文搜索。

## 如何添加文章？

### 方法一：通过 GitHub Issue（推荐）

1. 打开你的 GitHub 仓库
2. 点击顶部的 **Issues** 标签
3. 点击绿色 **New issue** 按钮
4. 选择 **"添加文章"** 模板
5. 粘贴公众号文章链接，填写信息
6. 点击提交，等待1-2分钟即可在网站看到

### 方法二：手动创建文件

在 `_posts` 目录下创建 Markdown 文件即可。

## 特性

- 📅 **按日汇总** - 每天的文章自动归为一组
- 🏷️ **标签管理** - 给文章打上标签，方便分类
- 🌙 **深色模式** - 点击右上角切换浅色/深色主题
- 📱 **响应式设计** - 手机、平板、电脑都能完美阅读
- 🔗 **原文跳转** - 每篇文章都可以一键跳转到原文

## 开始吧！

点击上方的 **"+ 添加文章"** 按钮，添加你的第一篇文章。
